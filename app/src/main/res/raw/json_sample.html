Editing events is done with a JSON file and looks like this.

{
    "twitchUrl": string(optional),
    "donateUrl": string(optional),
    "events": [
        {
            "time": long(milliseconds since 1970),
            "game": string,
            "runners": string,
            "est_time": string(h:mm:ss),
            "category": string,
            "casters": string,
            "setup_time": string(h:mm:ss)
        }
    ]
}
