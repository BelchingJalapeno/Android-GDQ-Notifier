Editing events is done with a JSON file and looks like this.

{
    "twitch_url": string(optional),
    "donate_url": string(optional),
    "events": [
        {
            "start_time": long(milliseconds since midnight, Jan 1st, 1970 in UTC),
            "game": string,
            "runners": string,
            "est_time": string(hh:mm:ss),
            "category": string,
            "casters": string,
            "setup_time": string(hh:mm:ss)
        }
    ]
}
